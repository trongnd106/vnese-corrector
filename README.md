# 1. Run Server Ulfius

## Run command to compile source

gcc server.c -o server -lulfius -ljansson -lmysqlclient

## Run command to start Server

./server

## Server Ulfius running on http://127.0.0.1:8080

# 2. Run Server Flask

## Run command to start Server

python3 server.py

## Server Flask running on http://127.0.0.1:8000
